- Install `ahooks` to use `useThrottleFn` for efficient scroll handling.
  - Command: `npm install ahooks`

### 2. Refactor `OverseasTab/index.tsx`

- **State & Refs**:

  - Implement the scroll listener using `useThrottleFn` (wait: 100ms).
  - Loop through `overseasCategories` to find the section currently at the top of the viewport.

  - Pass a custom `onChange` handler to `RegionSidebar` that finds the target anchor and calls `scrollIntoView()`.
