# Writing Skills

This skill helps you create and refine skills for the Trae agent.

## Instructions

1. Define the skill's name and purpose.
2. Write a clear and concise description that explains when to invoke the skill.
3. detailed instructions on how to perform the skill's task.
4. Include examples and edge cases if applicable.
5. Test the skill to ensure it works as expected.
