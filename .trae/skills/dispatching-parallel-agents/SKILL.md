# Dispatching Parallel Agents

This skill helps you execute multiple independent tasks in parallel.

## Instructions

1. Identify tasks that can be executed independently.
2. Launch separate agent instances for each task.
3. aggregate the results from all agents.
4. Ensure no conflicts or race conditions between the parallel tasks.
