# Writing Plans

This skill helps you create a structured plan for implementing a feature or task.

## Instructions

1. Analyze the requirements and identify the key components and steps.
2. Break down the task into smaller, manageable sub-tasks.
3. Define the dependencies and order of execution.
4. Create a checklist or todo list to track progress.
5. Review the plan to ensure it covers all requirements and edge cases.
