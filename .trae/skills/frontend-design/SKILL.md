# Frontend Design

This skill focuses on creating high-quality, visually appealing frontend interfaces.

## Instructions

1. Analyze the design requirements and user experience goals.
2. Choose appropriate UI components and layout patterns.
3. Apply consistent styling (colors, typography, spacing) following the design system.
4. Ensure responsiveness and accessibility.
5. Implement the design with clean, maintainable code.
