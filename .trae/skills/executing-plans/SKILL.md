# Executing Plans

This skill guides you through executing a pre-defined implementation plan.

## Instructions

1. Review the existing plan and current status.
2. Pick the next pending task from the plan.
3. Implement the task according to the specifications.
4. Verify the implementation (tests, manual check).
5. Update the plan status and move to the next task.
