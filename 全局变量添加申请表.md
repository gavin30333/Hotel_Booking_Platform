# 全局变量添加申请表

## 1. 颜色系统 (Color System)
| 变量名 | 值 | 推荐指数 | 用途 | 现状差异说明 |
| :--- | :--- | :--- | :--- | :--- |
| `@brand-blue` | `#0086F6` | ⭐⭐⭐ | 酒店业务线品牌主色 (评分/价格) | 与全局 `@primary-color` (`#0066ff`) 存在色差，建议独立定义 |
| `@brand-gold` | `#FFB400` | ⭐⭐⭐ | 品牌辅色 (钻石/高亮) | 与全局 `@warning-color` (`#faad14`) 不同，更偏暖金 |
| `@badge-bg-gold` | `#E0C08A` | ⭐⭐ | 金钻/高级徽章背景色 | 新增业务场景色 (原 `@muted-gold`) |
| `@badge-bg-ranking` | `#FEEEE3` | ⭐⭐ | 榜单徽章背景色 | 新增业务场景色 (原 `@ranking-bg`) |
| `@text-brown` | `#8B572A` | ⭐⭐ | 榜单/金钻深色文字 | 用于浅色金/粉背景上的深色对比文本 |
| `@text-tertiary` | `#999999` | ⭐⭐⭐ | 辅助信息/划线价 | 介于 `@text-secondary` (`#666`) 与 `@text-placeholder` (`#ccc`) 之间 |

## 2. 布局与形状 (Layout & Shapes)
| 变量名 | 值 | 推荐指数 | 用途 | 现状差异说明 |
| :--- | :--- | :--- | :--- | :--- |
| `@radius-xs` | `2px` | ⭐⭐ | 微圆角 (标签/评分框叶子角) | 全局最小为 `@border-radius-sm` (`4px`)，无法满足精致标签需求 |
| `@radius-img` | `6px` | ⭐⭐ | 图片专用圆角 | 介于 `@border-radius-sm` (`4px`) 与 `@border-radius-md` (`8px`) 之间 |
| `@font-size-xxs` | `9px` | ⭐ | 超小号字 (Tag标签内部) | 全局最小为 `@font-size-xs` (`10px`)，复杂标签内文字需更小 |
| `@font-size-11` | `11px` | ⭐⭐⭐ | 列表页/详情页核心信息基准字号 | 缺失关键中间档位，大量信息 (`10px` 太小看不清, `12px` 太大占空间) |
